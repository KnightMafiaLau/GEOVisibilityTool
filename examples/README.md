## 想贡献你的案例?

欢迎 PR。建议格式参考 `jianzhi-20260531/`:
- 一份 README.md(品牌信息、测试设置、关键发现、复现方式)
- 报告 PDF + 渠道指南 PDF(从原生 .html 浏览器 Cmd+P 导出)

发 PR 时请确认:
- 品牌信息属实(品牌名 / 行业 / 垂类 / 竞品)
- 报告中所有数据来自真实 LLM 实测,不是手工编造
- 案例公开发布**不会**对第三方造成不当影响(竞品评分基于公开 LLM 答案,但还是请负责任地展示)
